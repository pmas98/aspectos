REMEMBER Esse é um código pra rodar BABA SCRIPT

REMEMBER É assim que se atribui valores
BABA HAS 10;
APPLE HAS 5;

REMEMBER É assim que se define uma função
MAKE FUNCTION ADD_TWO X DO
    X IS X PLUS 2;
    SAY X;
BYE BYE

REMEMBER É assim que os loops funcionam
LOOP BABA IS NOT 0 DO
    SAY BABA;
    BABA IS BABA MINUS 1;
    USE ADD_TWO APPLE;
    APPLE IS APPLE PLUS 1;
BYE BYE

REMEMBER Essas são as condicionais
WHEN APPLE IS SAME AS 10 DO
    SAY "BANANA";
ELSE
    SAY "NOT YET";
BYE BYE
